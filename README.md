# 🏪 General Store Management System (GUI Edition)

A modern desktop-based **General Store Management System** built using **Python + CustomTkinter + SQLite**.

This application helps manage:
- Stock purchases
- Product sales
- Inventory tracking
- Transaction history
- Monthly analytics

Designed with a clean UI and structured database storage.

---

# ✨ Features

## 📦 Purchase Dashboard
- Add new stock
- Automatic date & time recording
- Auto total calculation
- Transaction confirmation

---

## 💰 Sales Dashboard
- Sell products directly

# 🚀 Installation

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
```

Move into folder:

```bash
cd GeneralStoreGUI
```

Install dependencies:

```bash
pip install customtkinter
pip install matplotlib
pip install pandas
pip install tkcalendar
```

Run:

```bash
python main.py
```

---

# 🗄️ Database

Transactions are stored using SQLite.

Database file:


- transactions
